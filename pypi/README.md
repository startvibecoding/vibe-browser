# vibe-browser

Fast browser automation CLI and Go SDK for AI agents. Each PyPI wheel includes a platform-native `vibe-browser` binary.

## Installation

```bash
pip install vibe-browser-installer
```

Or with an isolated environment:

```bash
pipx install vibe-browser-installer
```

After installation, the `vibe-browser` command is available on your `PATH`.

## Quick Start

```bash
vibe-browser open https://example.com
vibe-browser get title
vibe-browser screenshot --output page.png
vibe-browser snapshot --interactive
```

## Supported Platforms

- Linux x86_64, arm64
- Linux musl x86_64, arm64
- macOS x86_64, arm64
- Windows x64, arm64
